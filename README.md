## 🚀 Installation

1. Téléchargez `ChequierPro.zip` depuis la section **Releases**
2. Extrayez le dossier où vous le souhaitez
3. Double-cliquez sur **ChequierPro.exe**
4. Entrez votre clé de licence au premier lancement
